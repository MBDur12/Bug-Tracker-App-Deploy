About the Project

This is a basic bug tracker application to track bugs, changes, new features etc. in software projects.

Bugs are tracked using a ticket system, in which "submitters" submit new tickets corresponding to a current software project.

This web application was built primarily using the Django framework.

Roles

There are 4 distinct roles that users can have, each with different permissions and actions that they can perform. Below is a brief description of each role:

Submitter

  • Create new tickets for projects
  • View and edit tickets they have submitted

Developer

  • View and edit tickets they have been assigned to

Project Manager

  • View and edit tickets for projects they are assigned to
  • View and edit projects they are assigned to
  • View and archived projects
  • Create new projects
  • Manage users assigned to projects

Administrator

  • Assign user roles
  • View and edit all tickets and projects
  • Create new projects
  • Archive projects
